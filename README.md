# Flappy Math Bird - Projet IHM

Un mini-jeu interactive développé comme projet IHM (Interface Homme-Machine) pour la licence CNAM. Le projet combine le gameplay du célèbre jeu "Flappy Bird" avec des défis mathématiques en utilisant PHP pour la gestion des sessions.

## Fonctionnalités

- **Page de connexion** : Authentification utilisateur avec sessions PHP
- **Jeu interactif** : Mini-jeu de type Flappy Bird
- **Système de score** : Suivi et affichage des scores
- **Page d'accueil** : Interface principale du projet
- **Écran de fin** : Affichage quand le joueur perd

## Structure du projet

- `index.html` - Page de login/connexion
- `home.html` - Page d'accueil après connexion
- `score.html` - Affichage des scores
- `Mort.html` - Écran de fin de jeu
- `grosbec.jpg` - Image utilisée dans le jeu
- `Projet_IHM_MiniJeu_Session_PHP_L3.pdf` - Documentation du projet

## Technologies utilisées

- HTML5
- CSS3 (Tailwind CSS)
- JavaScript
- PHP (sessions)
- Material Design Icons

## Installation

1. Clonez le dépôt
2. Placez les fichiers dans le répertoire web de votre serveur (htdocs pour XAMPP)
3. Accédez au projet via votre navigateur
